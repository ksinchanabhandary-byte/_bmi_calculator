# ⚖️ Modern BMI Calculator
![App Screenshot]("Screenshot(1).png")

A sleek and interactive web application built with **Python** and **Streamlit** that allows users to calculate their Body Mass Index (BMI) using metric units or by converting feet to meters.
…

## 🚀 Features-
* Height Conversion: Automatically converts height from feet to meters for accurate calculations.
* Instant Classification: Categorizes results (Underweight, Normal, Overweight, Obese) based on WHO standards.
* Dynamic Feedback: Uses color-coded status boxes (Green, Yellow, Red) to display health categories.
* User-Friendly UI: Simple number inputs and a clean layout.
…

## Example Usage

  Once the app is running in your browser, follow these steps:

  Enter Weight: Input your weight in kilograms (e.g., 70).

  Enter Height: Move the slider or type your height in feet (e.g., 5.8).

  Check Result: The app will instantly display your BMI Score and a color-coded health category:

  🟡 Underweight (< 18.5)

  🟢 Normal (18.5 - 24.9)

  🟠 Overweight (25.0 - 29.9)

  🔴 Obese (> 30.0)
